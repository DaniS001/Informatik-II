func_picture.get_information(name, language)

Picture information

Reads the information of an image and returns it as a string. If the file does not exist, a default text in the appropriate language will be returned.

Parameters

  • name (str) – Name of the picture.

  • language (int) – Language in which the default message should be loaded.

Returns

Information of the picture

Return type

str
