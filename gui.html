class gui.MyComleteGUI(master='root')

Bases: tkinter.PanedWindow

Class for the main window of the GUI.

This Class creates the main window and all the features in it.

Parameters

  • state (str) – Shows the state of the clock. (offline / running / pause)

  • project_path (str) – Contains the path to the projectfolder.

  • master (tk) – Master is the parent for all widgets.

  • language (str) – Contains the language, which is loadet at the moment.

  • help_var_picture (int) – Contains which picture you can see at the moment.

  • picture_size (int) – Is responsible for the loadet picture size.

  • filepath (str) – The path to the last opened help.txt file.

start()

Starting of the stoppwath

The stopwatch starts. This sets the status to “running”, resets all time variables, and changes the button command. If the stopwatch status is “pause”, a new time is calculated for restarting. The button command is reset to stop. However, if the status is “running”, the function will repeat over and over again until stopped.
